사용자 안내서¶

이 사용자 안내서는 보케를 사용하기 위해 알아야하는 기본적인 사항들을 다루고 있다. 다음과 같은 순서로 구성된다.:

퀵스타트
초기 셋업 및 실행
Defining Key Concepts
보케의 구성에 대한 기본적인 중요 개념
Plotting with Basic Glyphs
bokeh.plotting 인터페이스가 제공하는 간단하고 유연한 글리프 메소드로 여러가지 플롯을 만들기
Providing Data for Plots and Tables
플롯과 테이블에 쓰이는 데이터 집합을 제공하고 공유하기
Laying out Plots and Widgets
특정 레이아웃에 따라 여러개의 플롯과 위짓을 합치기
Handling Categorical Data
바차트, 카테고리 히트맵, visual dodging, jitter 등의 여러가지 테크닉을 사용하여 카테고리 데이터를 다루
Visualizing Network Graphs
노드와 간선 상호작용을 보여주는 네트워크 그래프 시각화
Mapping Geo Data
Google Maps, GeoJSON, Tile Rendering 등 지형 데이터 다루기
Configuring Plot Tools
플롯에 사용할 수 있는 pan, zoom, select 등의 상호작용 도구
Styling Visual Attributes
axes, grids, labels, glyphs 등 보케 플롯의 모든 시각적 요소 설정하기
Adding Annotations
labels, arrows, legends 등의 정보 추가하기
Adding Interactions
위짓이나 연결 패닝, 선택 등 복잡한 상호작용 만들기
Running a Bokeh Server
복잡한 데이터베이스 애플리케이션을 위한 보케 서버 배포
Working in the Notebook
주피터 노트북에서 상호작용 플롯 표시하기
Exporting Plots
PNG, SVG로 보케 출력하기
Embedding Plots and Apps
스태틱 임베팅 또는 서버기반 보케 플롯과 위칫을 HTML문서에 넣기
Using bokeh Commands
커맨드라인에서 bokeh 명령으로 보케 기능 사용하기
Extending Bokeh
사용자 설정 기능을 추가하기
Leveraging Other Libraries
Datashader와 HoloViews 라이브러리를 보케와 같이 사용하기
Speeding up with WebGL
WebGL을 사용하여 대규모 데이터셋에 대한 성능 향상
Developing with JavaScript
BokehJS를 직접 사용하여 자바스크립트로 플롯 생성하기
Learning More
더 많은 정보와 예제

사용자 안내서의 예제들은 보케로 한가지 기능을 수행하도록 가능한한 간단하게 작성되었다. NumPy나 Pandas와 같은 외부 라이브러리의 도움은 거의 받지 않았다. 하지만 보케는 NumPy나 Pandas 또는 기타 테이블 형태의 데이터 구조와 같이 사용하기 좋다.
